# 🌊 Fhish Landing Page

This is the official landing page for **Fhish**, the privacy-preserving FHE (Fully Homomorphic Encryption) stack for the Initia ecosystem.

## 🚀 Overview
Fhish brings hardware-grade privacy to Initia using FHE. This landing page serves as the primary gateway for users and developers to learn about the protocol, access documentation, and launch the demo applications.

## 🛠️ Tech Stack
- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com)
- **Icons**: [Lucide React](https://lucide.dev)
- **Animations**: Custom CSS Mesh Gradients & Glassmorphism

## 🏃 Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Structure
- `src/app/`: Next.js pages and global styles.
- `public/`: Assets including the Fhish logo.
- `components/`: (If any) Reusable UI components.
